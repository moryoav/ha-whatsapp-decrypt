# Code of Conduct

## Our Pledge

We want WhatsApp Media Processor for Home Assistant to be a respectful and useful project for people with different Home Assistant setups, automation styles, and experience levels.

Contributors, maintainers, and users are expected to participate constructively in issues, pull requests, discussions, and any other project spaces.

## Expected Behavior

- Be kind, patient, and specific.
- Keep discussions focused on the project and the problem being solved.
- Assume good intent when possible.
- Respect different experience levels and Home Assistant environments.
- Remove secrets, private media URLs, media keys, OpenAI keys, private logs, and personal Home Assistant configuration before sharing examples.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks.
- Discriminatory language or behavior.
- Repeated disruption, trolling, or bad-faith arguments.
- Publishing another person's private information, logs, credentials, URLs, media keys, or Home Assistant configuration.
- Encouraging unsafe use of this project without clearly explaining the risk.

## Enforcement

Maintainers may remove comments, close issues or pull requests, limit participation, or take other reasonable action when behavior harms the project or its community.

If you see behavior that violates this Code of Conduct, report it to the project maintainer. If the issue involves a security vulnerability, follow `SECURITY.md` instead of posting details publicly.

## Scope

This Code of Conduct applies in all project spaces, including GitHub issues, pull requests, discussions, and any community spaces that represent the project.
